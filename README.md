# DUMB Android Client
> Written on React by debianrose in Termux :)

## How complie?
1. Install packages
``` shell
npm install
```
2. Build & Sync Project
``` shell
npm run build && npx cap sync
```
4. Go to `android` Folder
5. Complie with Gradle
``` shell
./gradlew assembleDebug
```

## Used Frameworks
### Capacitor - capacitorjs.com
> For Android Building :p
### Ionic Framework - ionicframework.com
> Material Design Like Interface
### Vite - vite.dev
> For Building and Starting up Web Version
### React - react.dev
> Yea not React Native!
